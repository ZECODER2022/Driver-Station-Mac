# Driver-Station-Mac
THIS IS NOT A OFFICAL FIRST ROBOTICS SOFTWARE - BUT A STUDENT MADE PORT FOR MACOS 26+

This is a barebones version of Driver Station for MacOS with the basic functions for connecting to and using robots that have a RoboRio.

**Features:**
- Connect to RoboRio
- Enable and Disable Robot
- E-STOP (No keybinds yet)
- Team Number
- Modes: TeleOp, Auto, and Test
- Console (Buggy - Doesn't work much)
- Restart and reboot RoboRio
- See Connected USB devices
- See battery Voltage
- Change Alliance (R1-3, B1,3)

This project is OpenSource
